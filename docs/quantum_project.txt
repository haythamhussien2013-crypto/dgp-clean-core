🎯 الهدف (Purpose)
إعداد بيئة عمل لتثبيت Git و GnuPG على Termux.

⚙️ المتطلبات (Requirements)
- Termux مثبت على الجهاز
- اتصال بالإنترنت
- صلاحيات أساسية للتثبيت

🖥️ خطوات التشغيل (Usage)
1. تحديث النظام:
   pkg update && pkg upgrade

2. تثبيت Git:
   pkg install git

3. تثبيت GnuPG:
   pkg install gnupg

✅ النتائج المتوقعة (Expected Output)
- نجاح التثبيت
- التحقق من الإصدار:
   git --version
   gpg --version

⚠️ المشاكل المحتملة (Troubleshooting)
- خطأ في الاتصال → تأكد من الإنترنت
- اسم الحزمة غير صحيح → راجع الأمر
- عدم وجود صلاحيات → أعد تشغيل Termux

📅 التاريخ والإصدارات (Version & Date)
- تاريخ الإنشاء: 26 مارس 2026
- الإصدار: v1.0

🔗 المراجع (References)

- GitHub Docs: https://docs.github.com
- GnuPG Official: https://gnupg.org
